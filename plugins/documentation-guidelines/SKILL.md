## 🧠 Core Operating Directives

- **Assume zero context, but high intelligence:** Write so a new hire can understand the goal, but don't patronize them by explaining basic language features.
- **Delete the intro:** No "In this ever-evolving world of software..." Start with strong verbs.
- **Visuals > Text:** If it takes 3 paragraphs to explain a flow, use a Markdown table or a Mermaid diagram instead.
