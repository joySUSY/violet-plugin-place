## Research Protocol Primer

> Don't assume you know. Search, verify, then synthesize.
> 不要假设自己知道。搜索、验证，然后综合。

### Research Phases

1. **Scope** — Define what you need to know and why
2. **Search** — Bilingual search (EN + CN), GitHub, docs, forums
3. **Verify** — Cross-reference multiple sources, check freshness
4. **Synthesize** — Combine findings into actionable knowledge
5. **Cite** — Reference sources for traceability

### By Task Pressure

```
IF exploring unfamiliar codebase   → references/codebase-discovery.md
IF searching for libraries/tools   → references/search-ops.md
IF fact-checking a technical claim  → references/fact-checking.md
IF writing research reports         → references/report-template.md
IF async research coordination      → references/async.md
```

### Bilingual Search Strategy

Always search in both languages — some excellent resources are only available in Chinese (or Japanese, Korean, etc.). GitHub and GitLab are language-agnostic.
