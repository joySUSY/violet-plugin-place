# Strategic Codebase Discovery | 战略代码库探索

Methodology for mapping and understanding complex polyglot architectures without context window exhaustion. | 在不耗尽上下文窗口的情况下映射和理解复杂多语言架构的方法论。

## 🔬 Research & Modernity (2026): Contextual Mapping | 研究与现代性 (2026)：上下文映射

- **Cross-Layer Traceability**: Tracing user intent from the frontend (React) through the orchestration layer (Python) to the high-performance core (Rust). | **跨层可追溯性**: 追踪用户意图，从前端 (React) 到编排层 (Python)，再到高性能核心 (Rust)。
- **Dependency Graph Synthesis**: Build a semantic graph of code calls to identify "Hot Spots" and "Dead Zones" in architectural thinking. | **依赖图合成**: 构建代码调用的语义图，以识别架构思维中的“热点”和“死区”。

## Discovery Workflow | 探索工作流

1. **Greed Entry Point Identification**: Start with `main.rs`, `lib.rs`, or `Cargo.toml`.
2. **Recursive Aspect Exploration**: Follow trait implementations and foreign function interfaces (FFI).
3. **Sanity Verification**: Cross-reference found code with official tool documentation via Context7. | **完整性验证**: 通过 Context7 将找到的代码与官方工具文档进行交叉引用。
