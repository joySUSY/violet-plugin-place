# Prime Rust Foundations

1. Read `SKILL.md` first for the global engine orientation.
2. Then read `references/foundations/INDEX.md`.
3. Read `references/foundations/rust-foundations-ownership-memory-safety.md`.
4. If ownership or borrow pressure is central, also read `references/foundations/rust-ownership-cookbook.md`.
3. Recover the current problem as one of these classes:
   - ownership / borrowing / lifetimes
   - type design / generics / traits
   - API ergonomics
   - baseline project structure
5. If the task also concerns structure, read `references/architecture/rust-architecture-and-scaffolding.md` and, if needed, `references/architecture/rust-architecture-decision-trees.md`.
6. State the selected Rust problem class and the next exact file to load.
7. Do not start coding until the problem class is explicit.
